# Portfolio Tracker
A simple python script that helps you track your stock portfolio using the [yfinance]library.
this tool allows you to add and remove stocks,view your portfolio's value, and visualize it's perfomance ovr time.

## Features
Add or remove stocks from your portfolio
Get real time portfolio valuation 
visualize portfolio perfomance over a custom time reange
summary view with current stock prices and values

### Usage
Clone the repository and run the script in your python environment then customize the main section to add your desired stock
